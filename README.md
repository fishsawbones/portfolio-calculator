# Portfolio Freedom Calculator

Interactive calculator showing how much capital is required to generate a target monthly income indefinitely, across four inflation/market scenarios.

**Live:** https://portfolio-calculator.vercel.app

## Features
- Four scenarios: Optimistic, Base Case, Pessimistic, Stress Test
- 50-year portfolio survivability chart with depletion markers
- Depletion timeline bar
- Sliders for return rate, tax drag, income target, sequence-of-returns buffer
- Purchasing power erosion chart

## Stack
Plain HTML/CSS/JS — no build step. Chart.js via CDN.

## Updating
Edit `index.html`, push to `main` branch. Vercel auto-deploys.
